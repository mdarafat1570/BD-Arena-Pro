A modern, feature-rich tournament platform where Bangladeshi gamers can compete in Free Fire and PUBG Mobile tournaments for real money prizes. Players can register, join tournaments, track their winnings, and withdraw via bKash or Nagad.

![BD Arena Pro Preview](https://via.placeholder.com/1200x630?text=BD+Arena+Pro+Preview)

## ✨ Features

### 🎮 Tournament Management
- **5 Active Tournaments** with varying formats (Solo, Duo, Squad)
- Real-time slot tracking with progress bars
- Tournament status indicators (Live, Open, Full)
- Entry fees and prize pool breakdown
- Per-kill bonus system

### 👥 User System
- User registration with username/password
- Secure login system
- Persistent user data using localStorage
- Welcome bonus of ৳250 for new users

### 💰 Wallet & Payments
- Virtual wallet balance management
- Add money functionality (৳500 increments)
- Total winnings tracking
- bKash and Nagad payment method selection
- Real-time balance updates

### 🏆 Leaderboard
- Top 5 players ranking system
- Display of matches played, wins, and total earnings
- Player avatars and UIDs
- Game-specific color coding (Free Fire/PUBG)

### 🎨 Modern UI/UX
- Dark theme with neon orange accents
- Animated background orbs and gradients
- Live notification ticker with tournament updates
- Responsive design for mobile and desktop
- Smooth animations and transitions
- Floating match cards on hero section
